Material 3 Theme Previewer Customize Material Design tokens in real-time

Info

This theme previewer demonstrates how Material 3 CSS variables can be dynamically changed at runtime.

Features:

  • ✨ Real-time theme updates
  • 🎨 Color customization
  • 📝 Typography controls
  • 📐 Shape & density settings
  • 🌓 Light/dark mode toggle
  • 📤 CSS export functionality

How it works:

Material 3 generates CSS custom properties (variables) that can be modified at runtime using JavaScript. This approach bypasses the compile-time limitation of Sass functions.
